1. I set up a password and username in docker-compose.yml first then run the command:

mc alias set myminio http://localhost:9000 <username> <password>


2. Created a bucket named mybucket by:

mc mb myminio/mybucket

3. I first created a file:

echo "Example for file" > testfile.txt

Then moved it to the mybucket by:

mc cp testfile.txt myminio/mybucket/

4. Created a mirror by adding the testfile.txt to a new dir called mirror_mybucket then made a couple more files same way as in 3. Then mirrored by using the command: 

mc mirror mirror_mybucket/ myminio/mybucket/

5. It will not be deleted as if we wanted it to be deleted we have to add the --remove command to our bucket. Like : 

mc mirror --remove mirror_mybucket/ myminio/mybucket/

Now it will work in a way that if we delete files in the dir then it will also delete in the bucket. Otherwise the default mirroring only will work when adding files not deleting and of course only when kind of update by mirroring again, if mirroring is not done then updates will not take place.
