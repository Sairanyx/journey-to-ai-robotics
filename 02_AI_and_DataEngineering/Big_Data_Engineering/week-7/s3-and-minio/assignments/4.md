# Part 1 – Setup Clean Environment

## 1️ Create Bucket

**Bucket name:**

company-data-2026

### MinIO
```bash
mc mb myminio/company-data-2026
```

### AWS
```bash
aws s3 mb s3://company-data-2026
```

### Enable Versioning
MinIO
```bash
mc version enable myminio/company-data-2026
```

AWS:
```bash
aws s3api put-bucket-versioning \
  --bucket company-data-2026 \
  --versioning-configuration Status=Enabled
```

Verify
```bash
mc version info myminio/company-data-2026
```


## Part 2 – Upload Clean Data
Create files
```bash
echo "Q1 Revenue: 1M" > report.txt
echo "Employee salaries confidential" > hr.txt
echo "Legal contract v1" > contract.txt
```

Upload files:
```bash
mc cp *.txt myminio/company-data-2026/
```
List objects
```bash
mc ls myminio/company-data-2026/
```

## Part 3 – Simulate Ransomware Attack
Simulate encryption attack
```bash
echo "YOUR FILES ARE ENCRYPTED. PAY 5 BTC." > report.txt
echo "YOUR FILES ARE ENCRYPTED. PAY 5 BTC." > hr.txt
echo "YOUR FILES ARE ENCRYPTED. PAY 5 BTC." > contract.txt
```

Upload again (overwrite)
```bash
mc cp *.txt myminio/company-data-2026/
```

## Observe

Download file:
```bash
mc cat myminio/company-data-2026/report.txt
```

You should see:

YOUR FILES ARE ENCRYPTED.

Answer to the following questions:

Is the original data gone?

Did we delete it?

## Part 4 – Investigate Versions
List versions
```bash
mc ls --versions myminio/company-data-2026
```

Students should see:

Old version (original)

New version (encrypted)

## Part 5 – Restore Clean Version
Option 1 – Copy old version to new file

```bash
mc cp myminio/company-data-2026/report.txt \
  --version-id <OLD_VERSION_ID> restored.txt
```

Option 2 – Restore directly
```bash
mc cp myminio/company-data-2026/report.txt \
  --version-id <OLD_VERSION_ID> \
  myminio/company-data-2026/report.txt
```

Verify
```bash
mc cat myminio/company-data-2026/report.txt
```

Data restored.

## Part 6 – Simulate Delete Attack
Attacker deletes object

```bash
mc rm myminio/company-data-2026/report.txt
```

List objects
```bash
mc ls myminio/company-data-2026
```
It appears deleted.

List versions again
```bash
mc ls --versions myminio/company-data-2026
```

You should see:

Delete marker
