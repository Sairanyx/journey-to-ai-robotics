# Data Analytics & Machine Learning

---

##  Course Overview

The course covers the full machine learning pipeline, from data preparation to model evaluation. Emphasis is placed on understanding why techniques are used, not just how to implement them.

The course follows:
- *Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow (3rd Edition)* by Aurélien Géron

---

## Key Concepts

###  Machine Learning Foundations
- What is Machine Learning and Data Analytics
- Supervised vs Unsupervised Learning
- Regression, Classification, Clustering

###  Data Handling
- Data exploration and visualization
- Data cleaning and preprocessing
- Feature engineering and transformation
- Handling categorical variables

###  Model Development
- Linear Regression, Logistic Regression
- Polynomial Regression
- K-Means Clustering
- Basic understanding of SVM and Random Forest

###  Training & Optimization
- Gradient Descent (batch, stochastic)
- Overfitting vs Underfitting
- Regularization (L1 / Lasso, L2 / Ridge)
- Bias-Variance trade-off

###  Model Evaluation
- Train / Validation / Test splits
- Cross-validation
- Metrics:
  - Accuracy
  - Precision
  - Recall
  - F1-score
  - Confusion Matrix
  - ROC Curve & AUC

###  Model Tuning
- Hyperparameter tuning
- Grid Search & Randomized Search

###  Clustering Evaluation
- Elbow Method
- Silhouette Score
