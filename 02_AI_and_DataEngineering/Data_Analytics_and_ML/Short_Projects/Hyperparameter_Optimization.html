Classwork 04: Wine Quality Prediction (Regression) with K-Fold Cross-Validation and Hyperparameter Optimization¶

Dataset Overview:¶

  • Name: Wine Quality Dataset (choose the Red Wine data) from UCI Machine Learning Repository
  • Link: https://archive.ics.uci.edu/ml/datasets/Wine+Quality
  • Description: Predict wine quality scores based on physicochemical tests.
  • Target Variable: quality (score between 0 and 10)
  • Features (11): Fixed acidity, volatile acidity, citric acid, residual sugar, chlorides, free sulfur dioxide, total sulfur dioxide, density, pH, sulphates, and alcohol.

Objective¶

You will train regression models using the Wine Quality Dataset, evaluate performance using 5-Fold Cross-Validation, and further optimize models using GridSearchCV and RandomizedSearchCV.

Problem Statement:¶

  • Predict the wine quality score based on chemical properties.
  • Evaluate results using RMSE.
  • Optimize hyperparameters using Grid Search and Random Search.

Tasks:¶

    • Load the dataset, handle missing values, and scale features.

    • Use GridSearchCV to tune hyperparameters for Decision Tree Regressor.

    • Report the best hyperparameters and RMSE scores.
  5. Final Model Evaluation
    • Select the best-performing model based on RMSE.
    • Evaluate and report RMSE on the test set for the best model.

    "min_samples_split": randint(2, 10)
}

from scipy import stats

# Set the confidence level (95%)

confidence = 0.95

# Calculating the squared errors (an array) between the final predictions and the true values

squared_errors = (y_test_pred - y_test) ** 2

# Computing the confidence interval for the mean squared errors using the t-distribution

np.sqrt(stats.t.interval(confidence, len(squared_errors) - 1,
                         loc = squared_errors.mean(),
                         scale = stats.sem(squared_errors)))

The RMSE measures the average prediction error between the predicted and the actual wine quality values. For example, if the mean wine quality is 5, our model will predict 0.55 off that 5 consistently, as in from 4.45 to 5.55. Considering the wine values range from 3-8 then 0.6 is an acceptable result for a model.

  4. . It means that if we provided the model with a different sample or repeated the the whole procedure many times, the confidence interval will be different each time. The confidence tells us the true average error of the model. If we repeated the process changing for example how we split the data for 100 times, about 95 times the interval would contain the true average RMSE.
  5. . The confidence interval of the Random Forest Regressor is 0.50102881 - 0.60488474.
  6. . The score of 0.5554 is acceptable and good considering of course the min and max values of the quality attribute ranges from 3-8. Additonally, the prediction error being under 1 quality point is a good indicator. Of course the smaller the range of the value of the attribute the worse our result will be considered to be, while the larger the better it will be considered. It is important to consider that the quality of the wine is of course initially somewhat subjective, so a perfect prediction is highly unlikely.
