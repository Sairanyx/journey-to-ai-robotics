# Embedded_Systems_Programming_UTU

## Overview
This course introduces the fundamentals of **embedded systems** and **C programming**.  
It focuses on how software interacts with hardware in real-world, resource-constrained systems.

## Learning Goals
- Understand what embedded systems are and where they are used
- Learn the basics of **C programming**
- Develop simple programs
- Understand system-level thinking (hardware + software)
- Work with simulations (Wokwi) instead of physical hardware
- Learn how to interact with peripherals and I/O

## Key Concepts
- Embedded systems = **task-specific, real-time, resource-limited systems**
- Programming = telling hardware what to do via software
- Computing stack:  
  `Problem → Algorithm → Code → Compiler → Hardware`

## Course Structure
- **Part 1:** C Programming Basics  
- **Part 2:** Embedded Systems Programming in C  

## Tools & Environment
- C programming (gcc)
- Linux / VM
- Wokwi simulator
- VS Code / IDE optional
