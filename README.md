## Research & Lab Experience

### 🔹 AIS Lab - Turku AMK - Summer Internship (SAVY Project)
**Period:** May–July 2025 (270 hours)  
**Role:** Research Intern — Autonomous & Intelligent Systems Laboratory, Turku AMK  
**Project:** Environmental monitoring platform for the Archipelago Sea drainage basin and coastal area

- Profiled and optimized a **Flask REST API** serving time-series sensor data from **InfluxDB**, identifying and documenting performance bottlenecks in query, write, and request times.
- Implemented **asynchronous data export workflows** using **Celery + Redis**, enabling reliable handling of large data requests that previously caused timeouts.
- Extended the API with **OpenAPI documentation, health monitoring routes**, and backend improvements including toast notifications on the **ReactJS** frontend.
- Worked across a full backend stack: **Python, Flask, PostgreSQL, InfluxDB, Docker, Linux/Ubuntu**, and GitLab version control.
- Delivered well-documented, production-ready code and received formal recognition from supervisor for initiative, reliability, and clear technical reporting.
